--- layout: default title: Welcome --- Welcome to Azure Distribution Bundle, a bundle to simplify deployment of Symfony application on Windows Azure. It solves the packaging for Azure, handles deployment of assets and distributed session management. Configuration for WindowsAzure is done through a new Symfony environment, so that your existing application can still be run in production mode without being coupled to Azure.
